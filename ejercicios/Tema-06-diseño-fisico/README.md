# Ejercicios Tema 6: Diseño Físico (DDL y DML)

## Descripción

Este directorio contiene los ejercicios prácticos correspondientes al Tema 6 del curso. El objetivo principal es aplicar el Lenguaje de Definición de Datos (DDL) para la construcción física de la base de datos, incluyendo la creación de esquemas, tablas, índices, columnas calculadas y restricciones de integridad.

## Archivos de Trabajo

A continuación se listan los scripts SQL que conforman esta práctica:

* **[01-creacion-base.sql](./01-creacion-base.sql):** Script inicial de arranque. Contiene la creación de la base de datos, la definición de los esquemas principales (CATALOGO, TRABAJADOR, PROYECTO, NOMINA) y las estructuras base.
