# Ejercicios Tema 7.2: Lenguaje de Consulta de Datos (DQL)

## Descripción

Este directorio contiene los scripts y resoluciones correspondientes a las prácticas del Tema 7.2 (DQL). El objetivo es dominar la extracción de información de la base de datos utilizando la instrucción `SELECT`, aplicando filtros, ordenamientos, agrupaciones (GROUP BY, HAVING), funciones de agregación y combinaciones de múltiples tablas (JOINs).

## Archivos de Trabajo

A continuación se listan los scripts SQL que conforman esta práctica:

* **[01-creacion-tabla-ejemplo.sql](./01-creacion-tabla-ejemplo.sql):** Script de preparación. Genera y puebla una tabla independiente llamada `empleadosEjemplo`, la cual es utilizada para probar consultas básicas, filtros de rango y correspondencia de patrones descritos en la teoría.
* **[02-resolucion-dql.sql](./02-resolucion-dql.sql):** Archivo de trabajo principal. Contiene la resolución de los 16 ejercicios integrales del tema, aplicando Joins, funciones de agregación y subconsultas sobre el modelo relacional completo de la empresa.
