# Tema 5 - Normalización de Bases de Datos

## Descripción

En esta carpeta se encuentran los ejercicios correspondientes al **Tema 5 de Bases de Datos**, enfocados en el proceso de **normalización de bases de datos**.

Los ejercicios consisten en analizar tablas y casos de estudio para identificar redundancias, dependencias funcionales y anomalías, aplicando las distintas formas normales hasta obtener un diseño más eficiente.

---

## Archivo de la carpeta

### 📄 ejercicicios_normalizacion.xlsx

Contiene diversos ejercicios prácticos para aplicar las técnicas de normalización de bases de datos.

### Actividades a realizar

- Identificar anomalías de inserción, actualización y eliminación.
- Analizar dependencias funcionales.
- Aplicar la Primera Forma Normal (1FN).
- Aplicar la Segunda Forma Normal (2FN).
- Aplicar la Tercera Forma Normal (3FN).
- Aplicar la Forma Normal de Boyce-Codd (FNBC).
- Aplicar la Cuarta Forma Normal (4FN).
- Descomponer tablas complejas en estructuras normalizadas.

---

## Ejercicios incluidos

1. **Anomalías**
   - Identificar problemas de redundancia y consistencia en las tablas.

2. **Dependencias funcionales**
   - Analizar las relaciones de dependencia entre atributos.

3. **Primera Forma Normal (1FN)**
   - Eliminar grupos repetidos y garantizar la atomicidad de los datos.

4. **Segunda Forma Normal (2FN)**
   - Eliminar dependencias parciales respecto a las llaves primarias.

5. **Tercera Forma Normal (3FN)**
   - Eliminar dependencias transitivas.

6. **Alumno - Tutor - Materia**
   - Caso práctico de normalización de información académica.

7. **Venta de autos**
   - Caso práctico para normalizar información de clientes, vehículos y ventas.

8. **Profesor - Materia - Textos**
   - Caso práctico para normalizar la relación entre profesores, materias y bibliografía.

9. **Venta de libros**
   - Caso práctico para normalizar información de libros, autores y librerías.

10. **Forma Normal de Boyce-Codd (FNBC)**
    - Aplicar restricciones adicionales a las dependencias funcionales.

11. **Cuarta Forma Normal (4FN)**
    - Resolver dependencias multivaluadas.
