# Participaciones

## Descripción

En esta carpeta se subirán las participaciones y ejercicios realizados a lo largo del curso.

Las actividades se irán agregando conforme avance el semestre y tendrán como objetivo reforzar los conceptos vistos en clase mediante ejercicios prácticos y actividades de aplicación.

## Contenido

Aquí podrán encontrarse:

- Ejercicios individuales.
- Actividades realizadas en clase.
- Participaciones asignadas por el docente.
- Material complementario relacionado con las actividades.

> Las evidencias se irán incorporando progresivamente durante el desarrollo del semestre.
