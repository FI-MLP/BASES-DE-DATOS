# Exposiciones

## Descripción

En esta carpeta se almacenarán los materiales y evidencias correspondientes a las exposiciones realizadas durante el curso.

Los temas de exposición serán sorteados en clase.

## Indicaciones

En el documento anexo se encuentran los detalles y requisitos que deben considerarse para la exposición.

Dos clases previas a la fecha asignada para la exposición, un integrante del equipo deberá subir todos los documentos correspondientes para su revisión.

El resto de los integrantes únicamente deberán entregar la tarea y colocar en comentario privado el nombre completo del compañero que realizó la entrega.

## Contenido

Aquí podrán encontrarse:

- Presentaciones.
- Material de apoyo.
- Documentos entregados para revisión.
- Evidencias relacionadas con cada exposición.

> Los materiales se irán agregando conforme se desarrollen las exposiciones durante el semestre.
