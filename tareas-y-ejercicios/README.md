# Tareas y Ejercicios

## Descripción

Este directorio está destinado a recopilar y almacenar las tareas y los ejercicios prácticos asignados a lo largo del curso de Bases de Datos. 

Las actividades documentadas en esta sección tienen como objetivo fundamental aplicar, desarrollar y consolidar los conocimientos teóricos adquiridos en clase. A través de la entrega progresiva de estos trabajos, se busca reforzar las habilidades analíticas y técnicas mediante la resolución de problemas concretos del área.

## Contenido

Dentro de este apartado se podrán encontrar las siguientes evidencias de trabajo:

* Tareas introductorias y de fundamentos teóricos.
* Ejercicios prácticos de diseño y modelado de datos (Modelo Entidad-Relación y MER Extendido).
* Prácticas de normalización estructural de bases de datos.
* Resoluciones de casos aplicados al álgebra relacional.
* Ejercicios de consultas SQL y manipulación de datos, incluyendo operaciones relacionales.
* Actividades y casos prácticos realizados de manera continua durante las sesiones de clase.

El material correspondiente a cada asignación se irá incorporando de forma progresiva conforme al avance del temario y el desarrollo del semestre académico.
