# Proyecto Final

## Descripción

En esta carpeta se almacenarán las evidencias y entregas correspondientes al proyecto final de la asignatura.

El proyecto se desarrollará de manera progresiva a lo largo del semestre mediante entregas parciales.

## Entregas

Se realizarán tres entregas parciales:

1. Primera entrega.
2. Segunda entrega.
3. Entrega final.

Cada entrega permitirá documentar el avance y desarrollo del proyecto hasta obtener la versión final.

## Contenido

Aquí podrán encontrarse:

- Documentación.
- Diagramas.
- Avances parciales.
- Entregables finales.
- Material complementario.

> Las entregas se irán incorporando conforme avance el semestre.
